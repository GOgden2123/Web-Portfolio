Instrument Store Homepage

Demo

What's the big idea?

Develop a instrument store landing page using the core concepts of JavaScript and the Vue framework while giving the user the ability to add and remove items from their cart. Also, help the user distingiush the different brands they can choose from by giving the items their own individual colors when hovered over.

StartingOut

Starting this project I began to code up the front end with HTML and Vue/CSS. I choose the two colors of navy blue for the background and bronze for the text due to it being a more unique combination and helped it stand out a little. I also decided to do different hue's of the required instrument brand colors when the user hovers over the different listings. Again, it's something that would help my project be different and stand out.

FinishingUp

After finishing the instrument store page I felt that I had a better understanding of JavaScript and how it worked, as well as how Vue works with JavaScript and can be beneficial to use later.
